### Reporting Bugs
* Check the [Issues tab](https://github.com/yourusername/africa_geo_gemini/issues) to see if the bug has already been reported.
* If not, open a new issue. Clearly describe the problem and include steps to reproduce it.

### Suggesting Enhancements
* We love new ideas! Whether it's a new geographic data point or a better AI prompt, please open an issue to discuss it.

### Pull Requests (PRs)
1. **Fork** the repository.
2. **Create a branch** for your feature (`git checkout -b feature/amazing-new-logic`).
3. **Write your code.** Ensure you follow the existing Ruby style (two-space indentation).
4. **Test your changes.** If you add a new geographic feature, ensure it passes the `Validator`.
5. **Commit your changes** (`git commit -m 'Add some amazing logic'`).
6. **Push to the branch** (`git push origin feature/amazing-new-logic`).
7. **Open a Pull Request**.
